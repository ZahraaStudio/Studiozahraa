# ZAHRAA STUDIO - متجر فاخر عصري

متجر إلكتروني احترافي وفاخر مع تصميم عصري (Glassmorphism)، سلة متطورة، ولوحة تحكم شاملة.

## ✨ الميزات الرئيسية

### 🎨 التصميم العصري
- **Glassmorphism**: تأثيرات زجاجية فخمة مع تدرجات ذهبية
- **Dark Luxury Theme**: تصميم أسود فاخر مع لمسات ذهبية
- **تجاوب كامل**: يعمل بشكل مثالي على الموبايل والتابلت والكمبيوتر
- **حركات سلسة**: انتقالات وتأثيرات دقيقة وأنيقة

### 🛍️ نظام المتجر
- **عرض المنتجات**: شبكة ديناميكية تتغير حسب حجم الشاشة (1 موبايل، 2 تابلت، 4 ديسكتوب)
- **سلايدر متقدم**: عرض متعدد الصور للمنتج مع نقاط التنقل
- **بحث فوري**: البحث عن المنتجات في الوقت الفعلي
- **تصفية ذكية**: تصفية المنتجات حسب الفئات

### 🛒 نظام السلة المتطور
- **سلة جانبية**: واجهة سهلة الاستخدام للسلة
- **حساب تلقائي**: حساب الإجمالي والضرائب تلقائياً
- **تحديث فوري**: تحديث الكميات والأسعار فوراً
- **حفظ محلي**: حفظ السلة في localStorage

### 📱 صفحة المنتج الاحترافية
- **سلايدر صور متقدم**: عرض الصور مع نقاط التنقل
- **معلومات مفصلة**: وصف شامل للمنتج
- **اختيار الكمية**: واجهة سهلة لاختيار الكمية
- **مشاركة سهلة**: مشاركة المنتج عبر الواتساب والشبكات الاجتماعية
- **منتجات ذات صلة**: عرض منتجات مشابهة

### 🎛️ لوحة التحكم الشاملة
- **إدارة المنتجات**: إضافة وتعديل وحذف المنتجات
- **إدارة الصور**: رفع صور متعددة لكل منتج
- **إدارة الأسعار**: تحديث الأسعار والأوصاف
- **تحكم كامل**: إدارة كل جوانب المتجر

### 📲 PWA (Progressive Web App)
- **تثبيت على الأجهزة**: تثبيت المتجر كتطبيق على الموبايل والكمبيوتر
- **عمل بدون إنترنت**: الوصول للمحتوى المخزن مؤقتاً
- **تحديثات تلقائية**: تحديث المحتوى تلقائياً عند التوفر
- **أيقونة مخصصة**: أيقونة فاخرة للتطبيق

## 🏗️ البنية التقنية

### التكنولوجيا المستخدمة
- **React 19**: مكتبة JavaScript الحديثة
- **TypeScript**: لغة برمجة آمنة من حيث النوع
- **Tailwind CSS 4**: إطار عمل CSS حديث
- **shadcn/ui**: مكتبة مكونات UI احترافية
- **Firebase**: قاعدة بيانات وتخزين سحابي
- **Wouter**: جهاز توجيه خفيف الوزن

### هيكل المشروع
```
client/
  ├── public/              # ملفات ثابتة (manifest، icons، service-worker)
  ├── src/
  │   ├── pages/          # صفحات التطبيق
  │   │   ├── Home.tsx    # الصفحة الرئيسية
  │   │   ├── Products.tsx # صفحة المتجر
  │   │   ├── Product.tsx # صفحة المنتج الفردي
  │   │   └── Admin.tsx   # لوحة التحكم
  │   ├── components/     # المكونات المعاد استخدامها
  │   │   ├── Header.tsx
  │   │   ├── ProductCard.tsx
  │   │   └── CartSidebar.tsx
  │   ├── hooks/          # الـ hooks المخصصة
  │   │   ├── useCart.ts
  │   │   └── useProducts.ts
  │   ├── contexts/       # React contexts
  │   ├── App.tsx         # المكون الرئيسي
  │   ├── main.tsx        # نقطة الدخول
  │   └── index.css       # الأنماط العامة
  └── index.html          # ملف HTML الرئيسي
```

## 🚀 البدء السريع

### التثبيت
```bash
# تثبيت الحزم
npm install

# بدء خادم التطوير
npm run dev

# البناء للإنتاج
npm run build
```

### الوصول إلى التطبيق
- **الصفحة الرئيسية**: http://localhost:3000/
- **المتجر**: http://localhost:3000/products
- **لوحة التحكم**: http://localhost:3000/admin
