# 🎯 متجر ZAHRAA STUDIO - الميزات الكاملة

## ✅ الميزات المطبقة بالكامل

### 1️⃣ واجهة المتجر (index.html)

#### 🏠 الصفحة الرئيسية
- عرض المنتجات المميزة
- شريط البحث
- قوائم التصنيفات
- نظام التقييمات والآراء

#### 🛍️ صفحة المتجر
- عرض جميع المنتجات
- تصفية حسب التصنيف
- عرض تفاصيل المنتج مع الصور (Swipe)
- اختيار الأنواع/المقاسات المختلفة
- زر "إضافة للسلة"
- زر "طلب عبر واتساب"

#### 🛒 السلة والدفع
- عرض المنتجات المضافة
- تعديل الكمية
- حساب الإجمالي
- تطبيق الكوبونات
- اختيار المحافظة (حساب رسوم الشحن تلقائياً)
- اختيار تغليف الهدايا
- الدفع عند الاستقبال

#### 👤 صفحة حسابي
- عرض وتعديل بيانات المستخدم
- حفظ الهاتف والمحافظة والعنوان
- عرض الطلبات السابقة
- عرض الكوبونات المخصصة
- إعدادات الإشعارات والحفظ التلقائي

#### 🎁 نظام الإحالة
- رابط إحالة فريد لكل مستخدم
- عرض عدد الإحالات
- عرض المكافآت المكتسبة
- حماية من الإحالة الذاتية
- حماية من تكرار الإحالة من نفس الجهاز

#### ⭐ نظام التقييمات
- إضافة رأي من المستخدم أو الضيف
- تقييم بالنجوم (1-5)
- عرض جميع الآراء
- حفظ في Firestore

#### 📦 تتبع الطلب
- إدخال رقم الطلب
- عرض حالة الطلب (قيد التنفيذ، تم الشحن، تم التوصيل، مكتمل، ملغي)
- عرض تفاصيل الطلب الكاملة

#### 🔐 تسجيل الدخول
- تسجيل الدخول عبر Google
- تسجيل الدخول عبر Facebook
- عرض اسم المستخدم وصورته
- زر تسجيل الخروج

#### 📞 أزرار التواصل
- 2 زر واتساب (للمتجر والعملاء)
- 1 زر فيسبوك
- تصميم Cards احترافي

---

### 2️⃣ لوحة التحكم (admin.html)

#### 🔑 تسجيل الدخول
- تسجيل دخول آمن بـ Email و Password
- عرض رسائل الخطأ

#### 📦 إدارة المنتجات
- إضافة منتج جديد
- تعديل المنتج
- حذف المنتج
- إضافة كود فريد لكل منتج (مثل: ZA-450)
- إضافة صور متعددة عبر Cloudinary
- إضافة أنواع/مقاسات مختلفة برسوم مختلفة
- عرض قائمة المنتجات

#### 🛒 إدارة الطلبات
- عرض جميع الطلبات
- تصفية الطلبات حسب الحالة
- تحديث حالة الطلب (قيد التنفيذ، تم الشحن، تم التوصيل، مكتمل، ملغي)
- عرض تفاصيل الطلب الكاملة
- عرض بيانات العميل
- عرض المنتجات المطلوبة
- عرض ملخص الحساب

#### 🏷️ إدارة الكوبونات
- إضافة كوبون جديد
- تحديد قيمة الخصم (مبلغ ثابت أو نسبة)
- تحديد الحد الكلي للاستخدام
- تحديد الحد لكل مستخدم
- تحديد تاريخ البدء والانتهاء
- عرض قائمة الكوبونات
- حذف الكوبون

#### 🎁 إدارة الإحالات والمكافآت
- تحديد قيمة المكافأة
- اختيار نوع المكافأة (مبلغ ثابت أو نسبة)
- تفعيل/إلغاء الإحالات
- عرض سجل الإحالات الناجحة

#### 🚚 إدارة الشحن
- تحديد أسعار الشحن لكل محافظة
- القاهرة، الجيزة، الإسكندرية، الشرقية، الدقهلية، المنوفية، الغربية، البحيرة

#### 🎉 إدارة العروض
- إضافة عرض يومي أو أسبوعي
- اختيار المنتج
- تحديد نسبة الخصم
- تحديد التاريخ
- عرض قائمة العروض
- حذف العرض

#### 📊 الإحصائيات
- عدد المنتجات الإجمالي
- عدد الطلبات الإجمالي
- عدد الزوار الإجمالي
- عدد الزوار الحاليين (أونلاين)
- أعلى المنتجات مبيعاً

#### ⚙️ الإعدادات العامة
- رقم واتساب المتجر
- سعر تغليف الهدايا
- رابط الفيسبوك
- رابط إنستغرام
- رسالة الترحيب
- حفظ جميع الإعدادات

---

### 3️⃣ نظام البيانات (Firebase)

#### 📦 Collections
- **products**: المنتجات
- **orders**: الطلبات
- **coupons**: الكوبونات
- **referrals**: الإحالات
- **reviews**: التقييمات والآراء
- **users**: بيانات المستخدمين
- **settings**: إعدادات المتجر
- **offers**: العروض

#### 🔒 الأمان
- تسجيل دخول آمن
- حماية البيانات من الوصول غير المصرح
- Device ID لحماية الإحالات

---

### 4️⃣ التكامل مع الخدمات الخارجية

#### ☁️ Cloudinary
- رفع الصور
- تخزين الصور
- الحصول على روابط آمنة

#### 🔐 Firebase Auth
- تسجيل الدخول بـ Google
- تسجيل الدخول بـ Facebook
- إدارة جلسات المستخدمين

#### 📱 WhatsApp API
- إرسال الطلبات عبر واتساب
- رابط واتساب مباشر

---

### 5️⃣ التصميم والواجهة

#### 🎨 التصميم
- تصميم احترافي بألوان ذهبية
- Responsive Design (يعمل على جميع الأجهزة)
- Tailwind CSS
- تأثيرات سلسة وانتقالات جميلة

#### 📱 الأجهزة المدعومة
- الهواتف الذكية
- الأجهزة اللوحية
- أجهزة الكمبيوتر

#### 🌙 الوضع الليلي
- تصميم مظلم احترافي
- ألوان مريحة للعين
