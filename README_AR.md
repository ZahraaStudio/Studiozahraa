# استوديو الزهراء - ZAHRAA STUDIO

متجر إلكتروني فاخر وعصري مع تصميم مميز، سلة متطورة، ودعم كامل لتطبيقات الويب التقدمية (PWA).

## ✨ الميزات الرئيسية

### 🎨 التصميم الفاخر
يتميز الموقع بتصميم عصري وفاخر مع خلفية سوداء وتفاصيل ذهبية، مما يعكس احترافية استوديو الزهراء. الواجهة تتجاوب بشكل كامل مع جميع أحجام الشاشات من الهواتف الذكية إلى أجهزة الكمبيوتر.

### 🛍️ نظام المتجر المتقدم
يوفر الموقع عرضاً ديناميكياً للمنتجات مع إمكانية البحث الفوري والتصفية حسب الفئات. يمكن عرض صور متعددة لكل منتج مع نقاط تنقل سهلة الاستخدام.

### 🛒 سلة المشتريات المحسّنة
تم تحسين تجربة السلة بإشعارات بسيطة وسلسة بدلاً من النوافذ المزعجة. يمكن إضافة وإزالة المنتجات بسهولة مع حساب تلقائي للإجمالي.

### 📸 ميزة مجموعات الصور للطباعة
يمكن إضافة عدة مجموعات صور في الطلب الواحد (مثل تابلوهات بمقاسات مختلفة). كل مجموعة لها اسم وصفي وتظهر منظمة في الطلب النهائي.

### 🖼️ عارض الصور المتقدم
انقر على أي صورة لفتحها بحجم أكبر وعرض أفضل. يعمل مع صور المنتجات وصور السلة والصور المرفوعة.

### 📱 دعم PWA الكامل
يمكن تثبيت الموقع كتطبيق على الهاتف والكمبيوتر. التطبيق يعمل بدون إنترنت ويتحدث نفسه تلقائياً.

### 🔍 تحسين SEO
اسم "استوديو الزهراء" يظهر في محركات البحث مع وصف دقيق. يتضمن الموقع ملفات robots.txt و sitemap.xml لتحسين الفهرسة.

## 🚀 البدء السريع

### الوصول إلى الموقع
- **الصفحة الرئيسية**: https://zahraa-studio.github.io/
- **المتجر**: https://zahraa-studio.github.io/#store-section

### تثبيت التطبيق
1. افتح الموقع في المتصفح
2. اضغط على قائمة المتصفح
3. اختر "تثبيت التطبيق"
4. سيظهر التطبيق على الشاشة الرئيسية

### استخدام مجموعات الصور
1. أضف منتجات للسلة
2. اضغط "إتمام الطلب"
3. في قسم "مجموعات الصور للطباعة"، اضغط "إضافة مجموعة صور جديدة"
4. أدخل اسم المجموعة (مثال: "تابلوهات 20x30")
5. اضغط "إضافة صور" واختر الصور
6. يمكن إضافة عدة مجموعات
