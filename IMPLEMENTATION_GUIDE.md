# 📋 دليل التطبيق - استوديو الزهراء

## 🎯 ملخص التحديثات

تم تطوير الموقع بنجاح مع إضافة الميزات التالية:

### ✅ الميزات المضافة

#### 1. دعم PWA الكامل
- تم تحديث `service-worker.js` ليشمل أيقونات التطبيق
- تم تحديث `manifest.json` مع اسم عربي كامل وأوصاف محسّنة
- إضافة اختصارات سريعة في التطبيق
- دعم العمل بدون إنترنت

#### 2. تحسين واجهة السلة
- إضافة نظام إشعارات بسيطة بدلاً من النوافذ المزعجة
- دالة `showToast()` تعرض إشعار بسيط عند إضافة/إزالة المنتجات
- الإشعارات تختفي تلقائياً بعد 3 ثواني

#### 3. ميزة مجموعات الصور للطباعة
- نظام `imageGroups` لتخزين مجموعات الصور
- دالة `addImageGroup()` لإضافة مجموعة جديدة
- دالة `renderImageGroups()` لعرض المجموعات بشكل منظم
- دالة `addImagesToGroup()` لإضافة صور لكل مجموعة
- دالة `removeImageFromGroup()` لحذف صور من المجموعة
- تظهر المجموعات في الطلب النهائي مع أسماء واضحة

#### 4. عارض الصور المتقدم
- إضافة نافذة `image-viewer-modal` لعرض الصور بحجم أكبر
- دالة `openImageViewer()` لفتح الصورة
- دالة `closeImageViewer()` لإغلاق الصورة
- يعمل مع جميع الصور في الموقع

#### 5. تحسين SEO واسم الموقع
- إضافة Meta Tags في `<head>` للوصف والكلمات الرئيسية
- إضافة Open Graph Tags لمشاركة أفضل على وسائل التواصل
- اسم "استوديو الزهراء" بالعربي في العنوان والوصف
- إنشاء ملف `robots.txt` لتوجيه محركات البحث
- إنشاء ملف `sitemap.xml` لخريطة الموقع

## 🔧 التفاصيل التقنية

### نظام الإشعارات
```javascript
function showToast(message) {
  const toast = document.createElement('div');
  toast.className = 'toast';
  toast.textContent = message;
  document.body.appendChild(toast);
  setTimeout(() => toast.remove(), 3000);
}
```

### مجموعات الصور
```javascript
let imageGroups = []; // تخزين المجموعات

// هيكل المجموعة:
{
  id: timestamp,
  name: "اسم المجموعة",
  images: ["base64_image1", "base64_image2"]
}
```

### عارض الصور
```javascript
function openImageViewer(imageSrc) {
  document.getElementById('viewer-image').src = imageSrc;
  document.getElementById('image-viewer-modal').classList.remove('hidden');
}
```

## 🚀 كيفية الاستخدام

### للعملاء:

#### تثبيت التطبيق:
1. افتح الموقع في المتصفح
2. اضغط على قائمة المتصفح (⋮ أو ≡)
3. اختر "تثبيت التطبيق" أو "Install app"
4. سيظهر التطبيق على الشاشة الرئيسية

#### استخدام مجموعات الصور:
1. أضف منتجات للسلة
2. اضغط "إتمام الطلب"
3. في قسم "مجموعات الصور للطباعة":
   - اضغط "إضافة مجموعة صور جديدة"
   - أدخل اسم المجموعة (مثال: "تابلوهات 20x30")
   - اضغط "إضافة صور" واختر الصور
   - يمكن إضافة عدة مجموعات

#### فتح الصور بحجم أكبر:
- انقر على أي صورة في المتجر أو السلة
- ستفتح الصورة بحجم أكبر
- انقر خارجها أو على ✕ للإغلاق

### للمسؤول:

#### عرض الطلبات مع مجموعات الصور:
في لوحة التحكم (admin.html)، ستظهر الطلبات مع حقل `imageGroups` يحتوي على:
```javascript
{
  id: timestamp,
  name: "اسم المجموعة",
  images: ["url1", "url2"]
}
```

## 🔍 اختبار الميزات

### اختبار PWA:
1. افتح DevTools (F12)
2. اذهب إلى Application > Service Workers
3. تحقق من أن Service Worker مسجل
4. اختبر العمل بدون إنترنت

### اختبار مجموعات الصور:
1. أضف منتج للسلة
2. اضغط "إتمام الطلب"
3. أضف مجموعات صور
4. تحقق من ظهورها في الطلب

### اختبار عارض الصور:
1. انقر على أي صورة
2. تحقق من فتح الصورة بحجم أكبر
3. اختبر الإغلاق
